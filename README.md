# Scroll Driven Interscroller

Minimal boilerplate for scroll-driven interscroller ad creatives. Optimized for iframe deployment with relative asset paths and single-bundle output.

```bash
# Install dependencies
npm install

# Start dev server (opens in browser with HMR)
npm run dev

# Build for production → ./dist
npm run build

# Preview production build locally
npm run preview

# Watch mode (rebuild on file changes)
npm run watch

# Format code with Prettier
npm run format

# Clean build artifacts
npm run clean
```
